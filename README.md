# Dima Fomin - Professional Sushi Chef Portfolio

A modern, professional landing page for Dima Fomin, a professional sushi chef looking for opportunities in Poland. Built as a Progressive Web App (PWA) using Next.js, shadcn/ui, and Framer Motion.

## üç£ Features

- **Full-screen Hero Section** - Eye-catching introduction with Dima's name and Polish tagline
- **About Section** - Professional bio highlighting experience in Japan and Europe
- **Portfolio Gallery** - Responsive grid of sushi creations with lightbox functionality
- **Skills Section** - Visual showcase of competencies with animated progress bars
- **Experience Timeline** - International career history with elegant timeline layout
- **Contact Form** - Functional contact form with social media integration
- **PWA Support** - Installable as a mobile app with offline capabilities
- **SEO Optimized** - Polish keywords and metadata for maximum visibility
- **Smooth Animations** - Framer Motion animations throughout
- **Japanese-Inspired Design** - Warm neutral colors and clean aesthetic

## üöÄ Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## üé® Customization

### Update Personal Information

1. **Hero Section** - Edit `components/sections/Hero.tsx`
2. **About Bio** - Modify `components/sections/About.tsx`
3. **Portfolio Images** - Update image URLs in `components/sections/Portfolio.tsx`
4. **Skills** - Customize skills array in `components/sections/Skills.tsx`
5. **Experience** - Edit work history in `components/sections/Experience.tsx`
6. **Contact Info** - Update social links in `components/sections/Contact.tsx`

### Add Real PWA Icons

Replace placeholder icons in `/public`:
- `icon-192x192.png` - 192x192px app icon
- `icon-512x512.png` - 512x512px app icon

You can create these using:
- [Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://www.pwabuilder.com/)

### SEO Metadata

Update SEO information in `app/layout.tsx`:
- Title and description
- Keywords for Polish job market
- Open Graph images
- Google verification code
