## 🔗 КАК ВСЕ РАБОТАЕТ МЕЖДУ СОБОЙ

### **1️⃣ ГОЛОВНА (Home Page - `/`)**
```
Что показывается:
├─ Hero секция "Modern Food Academy"
├─ Краткая инфо о платформе
├─ Курсы (preview)
├─ AI-Ментор (промо)
├─ ChefTokens (система)
└─ CTA: [Начать обучение] → /academy

Пользовательский путь:
Новичок → Видит красивую страницу → Кликает "Начать" → /academy
```

---

### **2️⃣ АКАДЕМІЯ (`/academy`)**
```
Что находится:
├─ Список всех курсов (50+ готовых)
├─ Фильтры по уровню
├─ Каждый курс показывает:
│  ├─ Название, описание
│  ├─ Уровень (Начинающий/Средний/Продвинутый)
│  ├─ Цена в ChefTokens
│  ├─ Рейтинг (4.9⭐)
│  ├─ Прогресс (45% завершено)
│  └─ [Начать/Продолжить] button
│
├─ После выбора курса → /academy/courses/[id]
│  ├─ Видео-уроки
│  ├─ Практические задания
│  ├─ AI-помощь в реальном времени
│  └─ [Завершить курс] → +1000 ChefTokens → Профиль обновлен

└─ Боковая панель: Лидерборд (кто больше заработал)

СВЯЗЬ С ДРУГИМИ:
├─ Если в курсе нужны рецепты → Ссылка на /market
├─ Если нужна помощь → "Спросить у AI" → /chat/create-chat
├─ Прогресс сохраняется в профиль → /profile
└─ Заработанные токены → /profile (кошелек)
```

---

### **3️⃣ МАРКЕТ (`/market`)**
```
Что находится:
├─ Каталог рецептов (авторские от шефа + пользователей)
├─ Каждый рецепт показывает:
│  ├─ Название, фото
│  ├─ Цена в ChefTokens (10-100)
│  ├─ Автор (шеф, другой пользователь, или AI-рецепт)
│  ├─ Рейтинг, отзывы
│  ├─ Ингредиенты, шаги
│  ├─ Food Pairing (с какой пить)
│  └─ [Купить] или [Добавить в избранное]

После покупки:
├─ Рецепт добавляется в "Мои рецепты"
├─ Можно использовать в холодильнике
├─ Можно поделиться с друзьями
└─ ChefTokens списываются со счета

СВЯЗЬ С ДРУГИМИ:
├─ Рецепты создаются на основе курсов → /academy
├─ AI помогает выбрать рецепт → /chat/create-chat
│  "У меня есть X продукты, что готовить?" → AI генерирует
├─ Добавленные рецепты появляются в холодильнике → /fridge
├─ История покупок в профиле → /profile (история расходов)
└─ Авторы получают ChefTokens → их профиль
```

---

### **4️⃣ AI-НАСТАВНИК (`/chat/create-chat`)**
```
Что находится:
├─ Главный интерфейс чата с Dima Fomin AI
├─ История сообщений
├─ Возможности:
│  ├─ Спросить что угодно о готовке (бесплатно/дешево)
│  ├─ Загрузить фото блюда (10-20 токенов)
│  │  └─ AI анализирует: подача, текстура, техника
│  ├─ Запросить рецепт (15-30 токенов)
│  │  └─ AI генерирует полный рецепт + паринг
│  ├─ Получить видео-рекомендацию (5 токенов)
│  │  └─ Ссылка на курсы /academy или материалы
│  └─ Live помощь при готовке (20-50 токенов)

СВЯЗЬ С ДРУГИМИ:
├─ AI может рекомендовать курсы → /academy/courses/[id]
├─ AI может предложить рецепт → /market (поиск или генерация)
├─ AI может подсказать ингредиенты → /fridge
├─ История вопросов хранится в профиле → /profile
└─ Расходы токенов отражаются в кошельке → /profile (чат-история)
```

---

### **5️⃣ ХОЛОДИЛЬНИК (`/fridge`)**
```
Что находится:
├─ Мой список ингредиентов (что есть дома)
├─ Добавить/удалить ингредиент
├─ Для каждого ингредиента:
│  ├─ Количество
│  ├─ Дата истечения
│  └─ Норма калорий (для профиля здоровья)

После добавления ингредиентов:
├─ [Что готовить из этого?] → Идет в AI-наставник
│  └─ AI генерирует подходящие рецепты (за токены)
│
├─ Автоматически предлагаются рецепты → /market
│  └─ "Из вашего холодильника можно готовить..."
│
└─ Интеграция с профилем здоровья
   └─ Калории считаются в профиле

СВЯЗЬ С ДРУГИМИ:
├─ AI помощь при выборе рецепта → /chat/create-chat
├─ Поиск рецептов по ингредиентам → /market
├─ Калории учитываются в здоровье → /profile (профиль здоровья)
└─ История использования в профиле → /profile
```

## 🔄 ПРИМЕРЫ ПОЛЬЗОВАТЕЛЬСКИХ ПУТЕШЕСТВИЙ

### **ПУТЬ 1: Новичок хочет научиться готовить суши**

```
1. Посещает сайт → ГОЛОВНА
   ├─ Видит красивый дизайн
   └─ [Начать обучение] → АКАДЕМІЯ

2. АКАДЕМІЯ
   ├─ Выбирает курс "Nowoczesna Kuchnia: Podstawy"
   ├─ Смотрит видео-уроки
   ├─ На середине урока: "Как правильно резать рыбу?"
   └─ [Спросить AI] → ЧАТ

3. ЧАТ
   ├─ Отправляет фото своего ножа и рыбы
   ├─ AI дает советы (-15 токенов)
   ├─ После: "Хочешь рецепт?" → Генерирует рецепт (-20 токенов)
   └─ "Смотри вот этот курс для продвинутых" → Ссылка на АКАДЕМІЯ

4. МАРКЕТ (опционально)
   ├─ Ищет "рецепты суши"
   ├─ Видит авторский рецепт шефа Dima Fomin
   ├─ Покупает (-50 токенов)
   └─ Добавляется в "Мои рецепты"

5. ХОЛОДИЛЬНИК
   ├─ Добавляет ингредиенты дома
   ├─ "Что готовить?" → ЧАТ
   └─ AI предлагает рецепты из его ингредиентов

6. ПРОФИЛЬ
   ├─ Видит прогресс: Уровень 3, 1500 XP
   ├─ История: -150 токенов потрачено, +500 заработано на курсе
   ├─ Здоровье: Калории из рецепта считаются автоматически
   └─ "Поздравляем! Вы на полпути к уровню 4"
```

---

### **ПУТЬ 2: Опытный пользователь продает свой рецепт**

```
1. ПРОФИЛЬ
   ├─ "Я создал отличный рецепт суши Rainbow"
   └─ [Загрузить авторский рецепт] → МАРКЕТ

2. МАРКЕТ
   ├─ Создает новый рецепт с описанием, фото, ингредиентами
   ├─ Устанавливает цену: 35 ChefTokens
   └─ [Опубликовать]

3. Другие пользователи видят его рецепт
   ├─ Рейтинг повышается
   ├─ Каждая покупка: +35 токенов ему в ПРОФИЛЬ
   └─ Появляется в лидербордe авторов

4. ПРОФИЛЬ (его)
   ├─ Заработок от продаж: +35, +35, +35...
   ├─ Уровень повышается
   └─ Может купить премиум-курсы на заработки
```

---

### **ПУТЬ 3: Пользователь готовит по рецепту с помощью AI**

```
1. МАРКЕТ → Купил рецепт "Приготовление унаги"

2. ХОЛОДИЛЬНИК → Добавил ингредиенты

3. ЧАТ → "Я начал готовить. Помощь пошагово?"
   ├─ AI: "Шаг 1: Режь унаги вот так..."
   ├─ Пользователь: "Готово, и дальше?"
   ├─ AI: "Шаг 2: Обработай терпически при 180°"
   ├─ ... (live помощь во время готовки)
   ├─ Пользователь: "Готово! Проверь фото"
   ├─ AI: "Отлично! 8/10. Вот советы для улучшения"
   └─ Стоимость сессии: -40 токенов

4. ПРОФИЛЬ
   ├─ История расходов: -40 токенов за сессию
   ├─ Калории: +150 ккал (добавлено в здоровье)
   └─ Достижение: "Готовил с помощью AI"
```

## 🔐 ДОСТУП & ЛОГИКА

```
Не залогинен:
├─ Может смотреть ГОЛОВНА
├─ Может просмотреть АКАДЕМІЯ (урезано)
├─ Может просмотреть МАРКЕТ (урезано)
└─ ЧАТ, ХОЛОДИЛЬНИК, ПРОФИЛЬ заблокированы → "Залогиньтесь"

Залогинен:
├─ ПОЛНЫЙ доступ ко всему
├─ Все данные синхронизируются между разделами
└─ История действий сохраняется в БД
```
